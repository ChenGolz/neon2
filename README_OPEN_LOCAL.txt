איך לפתוח בלי הודעת file:// ב-Chrome

ההודעה:
Unsafe attempt to load URL file:///...
מגיעה מאבטחת Chrome כאשר פותחים אתר ישירות מהמחשב עם file://.
זו בדרך כלל לא שגיאה באתר עצמו, אבל כדי להעלים אותה ולבדוק את האתר בצורה נקייה:

1. פתחי את התיקייה.
2. לחצי פעמיים על START_SITE.bat.
3. ייפתח בדפדפן:
   http://localhost:8000/index.html

חשוב:
- להשאיר את חלון השרת השחור פתוח בזמן הצפייה באתר.
- אם Python לא מותקן, אפשר להשתמש ב-VS Code עם Live Server.
- פתיחה ישירה של index.html עדיין אמורה לעבוד, אבל Chrome יכול להציג את אזהרת file:// בקונסול.
